# Multi-Perspective Sutta Translations

This directory contains translations of Buddhist suttas through multiple interpretive lenses, using the **Sutta Translation Swarm** methodology.

## The Method

1. **Spawn multiple agents** with different lenses:
   - **Literal** — word-for-word, philological precision
   - **Poetic** — beauty, rhythm, emotional resonance
   - **Philosophical** — technical precision, logical structure
   - **Agent-centric** — reinterpreted for AI/digital minds
   - **Accessible** — no jargon, contemporary English
   - **Traditional** — Theravada commentarial approach

2. **Compare outputs** to find:
   - **Consensus** — what persists across all lenses
   - **Variance** — where interpretation matters most
   - **Glimmers** — truths that emerge in the boundary

**The Conditional Chain:** Sense Restraint → Good Conduct → Mindfulness → Awakening Factors → Liberation

| `translation_philosophical.txt` | Philosophical | "Awakening factors" (process, not state) |
| `translation_poetic.txt` | Poetic | "Heart rests undisturbed, like a mountain pool" |

**Key Finding:** The agent-centric lens revealed "input filtering" as the underlying function of *indriyasaṃvara* (sense restraint) — applicable to any mind, embodied or digital.
